# Smart-Appointment-Booking-project
Manual appointment booking leads to double bookings, confusion, and poor  user experience. The goal is to build a full-stack appointment booking system  that ensures conflict-free scheduling.

How to Run the Project

Follow the steps below carefully:

Step 1: Open Project in VS Code
Open VS Code
Select and open the project folder

Step 2: Install & Start Server
Open the Terminal in VS Code
Run the following command:
npm start
The server will start running successfully

Step 3: Open the Application
Minimize VS Code
Open the project folder in File Explorer
